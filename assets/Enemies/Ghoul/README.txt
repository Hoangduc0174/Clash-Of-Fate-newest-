Ghoul Enemy | Pixel Monsters Vol. 4  by elesrech
==============================================

Contents
------------------

This asset pack features a fully animated ghoul enemy designed for platformer and action games.

The character includes 7 animations (44 frames):

	-Idle (10 frames)
	-Walk (6 frames)
	-Jump / Land (2 / 1 frames)
	-Attack (10 frames)
	-Hurt (5 frames)
	-Death (7 frames)
	-Bone projectile (3 frames)
		
Technical details.
------------------

- Character size: Approximately 24x24
- Canvas size: 80x80
- Format: PNG files, provided both in spritesheets and individual frames.
- Outline version only.
