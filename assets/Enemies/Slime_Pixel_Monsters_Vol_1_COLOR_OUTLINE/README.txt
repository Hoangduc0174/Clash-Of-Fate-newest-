Slime Enemy | Pixel Monsters Vol.1 by elesrech
==============================================

Contents
------------------

This asset pack features a fully animated slime enemy designed for platformer and action games.
The character includes 11 animations (95 frames):

	-Idle (8 frames)
	-Walk (6 frames)
	-Bounce (8 frames)
	-Slide start (7 frames)
	-Slide loop (6 frames)
	-Attack 1 (12 frames)
	-Attack 2 (13 frames)
	-Hurt 1 (5 frames)
	-Hurt 2 (5 frames)
	-Death (10 frames)
	-Revive (7 frames)

All come in 7 color variations: green, blue, purple, cyan, silver, red & yellow.


Technical details.
------------------

- Character size: approximately 24x24.
- Canvas size: 80x80
- Format: PNG files, provided both in spritesheets and individual frames. 
- Contains black & colored outlines.
- Contains 7 color variations: green, blue, purple, cyan, silver, red & yellow.
